# 1137. 第 N 个泰波纳契数列

泰波那契序列 $T_n$ 定义如下： 
$$
T_{0}=0, T_{1}=1, T_{2}=1, \text { 且在 } n>=0 \text { 的条件下 } T_{n+3}=T_{n}+T_{n+1}+T_{n+2}
$$
给你整数 `n`，请返回第 $n$ 个泰波那契数 $T_n$ 的值。

**示例 1：**

```()
输入：n = 4
输出：4
解释：
T_3 = 0 + 1 + 1 = 2
T_4 = 1 + 1 + 2 = 4
```

**示例 2：**

```()
输入：n = 25
输出：1389537
```

**提示：**

- $0 \leqslant \mathrm{n} \leqslant 37$
- 答案保证是一个 32 位整数，即 `answer <= 2^31 - 1`。
